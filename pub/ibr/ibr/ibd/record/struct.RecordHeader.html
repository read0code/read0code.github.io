const REC_INFO_MIN_REC_FLAG: u8 = 16u8

Info bit denoting the predefined minimum record: this bit is set if and only if the record is the first user record on a non-leaf B-tree page that is the leftmost page on its level (PAGE_LEVEL is nonzero and FIL_PAGE_PREV is FIL_NULL).

source

const REC_INFO_DELETED_FLAG: u8 = 32u8

The deleted flag in info bits; when bit is set to 1, it means the record has been delete marked

source

const REC_INFO_VERSION_FLAG: u8 = 64u8

Use this bit to indicate record has version

source

const REC_INFO_INSTANT_FLAG: u8 = 128u8

The instant ADD COLUMN flag. When it is set to 1, it means this record was inserted/updated after an instant ADD COLUMN.
